# Samples

Runnable examples built on top of the Agentic Retrieval project.

| Sample | Description |
| --- | --- |
| [QA_CLI](QA_CLI/) | Interactive terminal app: ask a question and compare **Tool-use**, **Agentic Retrieval** (decomposed), and a single-shot **vector search** baseline — or run all three side by side. |

Each sample is self-contained under its own folder and reuses the project modules
in the repository root without modifying them.
